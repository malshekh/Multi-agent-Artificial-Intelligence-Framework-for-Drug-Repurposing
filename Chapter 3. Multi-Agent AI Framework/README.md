## What is and what is not an agent?
I will start by defining what an agent is to better understand what I will be developing via python. First and foremost, the standard question and answer chat bot is *not* an agent. What makes an application agentic is its ability to take action autonomously without the need for human intervention, like reading an email, searching the internet, or writing a block of code, preceive the output of those actions and react accordingly. In that sense, you can say that chatbots have been getting more agentic, ChatGPT searches the web, Gemini reads the spreadsheets you've uploaded. 

I followed LangChain's course titled ['Foundation: Introduction to LangChain - Python'](https://academy.langchain.com/courses/take/foundation-introduction-to-langchain-python/texts/71234844-getting-set-up-text). I used particular models and model providers including ChatGPT, Gemini, Tavily, Anthropic and LangSmith.
